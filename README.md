# cli-lane

`cli-lane` is a small Bun-based CLI scaffold for local coordination work.

## Goal

This repository currently only establishes the minimal project skeleton. It does not yet include server, client, or task-manager modules.

## Development

- `bun run src/cli.ts --help`
- `bun run src/cli.ts`
- `bun test`
- `bun run build`

## Scope

Current scope is limited to a local coordination CLI.
